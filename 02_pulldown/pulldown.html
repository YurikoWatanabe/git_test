<select>タグについて

プルダウンメニューを作るため、
まずはformタグの中にselectタグとoptionタグを
書きましょう！

何ですかそれ？

今回用意したメニューは3つ。
1番目の選択肢はこのページ自身、
2番目の選択肢はさる氏のプロフィールページ、
3番目の選択肢は私うさのプロフィールページです。！

pulldown.html 
    <form id="form">
        <select name="select">
            <option value="pulldown.html">プルダウンメニュー</option>
            <option value="pulldown_saru.html">さるプロフィール</option>
            <option value="pulldown_usa.html">うさプロフィール</option>
        </select>
    </form>

selectタグのname属性はプルダウンメニューの名前、
optionタグのvalue属性は選択肢の値（ファイル名）です。

イベントの設定

jsファイルで、イベントの設定をしましょう！
.onchangeイベントは、フォームに入力された内容
が変わったときに発生します。

書式　onchangeイベント 取得した要素.onchange = function() {

    //イベント発生時の処理

};

フォーム要素の部品を取得するには、
フォーム部品のname属性を指定します。

書式　getElementByIdでform要素の部品を取得する場合 document.getElementById('formタグのID名').selectタグのname属性.onchange = function() {

    //イベント発生時の処理

};

//イベント発生時に行なう処理のところに
「選択されているoptionタグのvalue属性を調べ、URLを書き換える」
と記述することで、ページが切り替わります。

optionタグのvalue属性の値を取得するには、
selectタグのvalueプロパティを読み取ります。
